description: Build the Disk Analyzer application
allowed-tools: Bash
---

Build the DiskAnalyzer macOS application using xcodebuild.

You should:
1. Run xcodebuild with the DiskAnalyzer scheme in Debug configuration
2. Display the build status and any errors
3. Confirm successful build or report issues

```bash
xcodebuild -scheme DiskAnalyzer -configuration Debug build
```
